# Camel Up

A digital version of the Camel Up board game built with Python, Pygame, and MySQL.

## Setup

1. Clone repo
2. Create venv: `python -m venv venv`
3. Activate venv
4. `pip install -r requirements.txt`
5. Copy `.env.example` to `.env` and configure your MySQL credentials.
6. Create the MySQL database: `CREATE DATABASE camel_up;` (Optional: The application auto-creates missing DB schema on the first run).
7. Run the game: `python main.py`
